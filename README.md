# Polytube - E-Commerce Website (Laravel) 🛒📦

Polytube is a **web-based E-Commerce system** developed using **Laravel, PHP, and MySQL**.
It allows users to browse products, view details, and place orders while administrators can manage products, categories, and website content.

---

## 🔥 Features

### 🧑‍💻 Admin Panel

* Add, update, and delete product categories
* Add, update, and delete products
* Manage product images
* View and manage customer orders
* Activate or deactivate products
* Dashboard for managing website content

---

### 👤 User Panel

* Browse all product categories
* View product details
* Search for products
* Add products to cart
* Place orders easily
* Responsive user interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/AditiPatel-77/Polytube-E-Commerce-Website--in-Laravel.git
```

### Navigate to Project Folder

```bash
cd Polytube-E-Commerce-Website--in-Laravel
```

### Install Dependencies

```bash
composer install
```

### Create Environment File

```bash
cp .env.example .env
```

### Generate Application Key

```bash
php artisan key:generate
```

### Run Database Migration

```bash
php artisan migrate
```

### Start Development Server

```bash
php artisan serve
```
